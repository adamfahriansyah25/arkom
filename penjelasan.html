üóÇÔ∏èApa itu Register?

Register adalah tempat penyimpanan sementara di dalam processor (otak komputer). Seperti tangan robot yang memegang sesuatu sementara waktu.

  • AL = Register kecil (8 bit) untuk menyimpan data sementara
  • DX = Register untuk menyimpan alamat port yang akan diakses

üîåApa itu Port?

Port adalah pintu komunikasi antara processor dengan perangkat luar (sensor, aktuator, dll). Seperti colokan listrik di dinding rumah.

  • Port A (0080H) = Pintu untuk mengirim sinyal ke AC/Heater
  • Port B (0082H) = Pintu untuk membaca sensor suhu
  • Port C (0084H) = Pintu untuk membaca tombol ON/OFF

4Ô∏è‚É£ Instruksi CMP (Compare = Bandingkan)

Kode Penjelasan Teknis Analogi Sederhana

Instruksi ini TIDAK mengubah nilai AL, hanya membandingkan dan menyimpan hasil perbandingan secara internal

Hasil perbandingan digunakan untuk:
- JE (Jump if Equal) = Lompat jika sama
- JGE (Jump if Greater or Equal) = Lompat jika lebih besar atau sama
- JLE (Jump if Less or Equal) = Lompat jika lebih kecil atau sama
üéØ Analogi:
"Robot, lihat angka yang kamu pegang (AL). Bandingkan dengan angka 17. Apakah sama? Lebih besar? Lebih kecil?"

Seperti Anda timbang badan:
- Berat Anda = AL
- Berat ideal = 17
- Timbangan cuma ngasih tau: lebih berat, lebih ringan, atau pas
- Tapi berat Anda TIDAK berubah!
